CSS - bygg webben med stil

CSS används för att beskriva utseendet hos innehåll på webben. Denna kursen går igenom grunderna, från positionering till selektorspecificitet. Vi tittar på hur renderingsmotorer och DOM påverkar, samt hur man kan skriva prestandavänlig, skalbar CSS.

Kursen fokuserar också på hur man strukturerar och separerar CSS för ett ökat fokus på hanterbarhet i mycket stora applikationer.

Vi tar en titt på hur CSS3 kan driva våra webbplatser med effekter såsom bildtransparens och animationer, och hur mycket stöd för dessa features vi kan förvänta av dagens web.

Vi avslutar dagen med att titta på hur man kan lägga ytterligare djup till våra webbplatser genom att kombinera styrkan i CSS selectors i jQuery, ett kraftfullt JavaScript-bibliotek för att göra DOM-förändringar.

Målgrupp

Kursen är avsedd för mjukvaruutvecklare som önskar en introduktion till CSS, samt hur man använder CSS i praktiken med moderna metoder.
